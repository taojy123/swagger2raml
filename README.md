# swagger2raml
convert swagger json data to raml
=============================


Usage
----------------------------------

    $ git clone https://github.com/taojy123/swagger2raml
    $ cd swagger2raml
    $ python s2r
    Finish!
    $


Or add custom argvs:

    $ python s2r MY_API http://heidianapi.com/api/docs/api-docs/ ./result
    Finish!
    $

Then go to see result dir.
